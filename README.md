## 🎯 Objetivo

O **Café Aurora** é um projeto fictício desenvolvido para portfólio, com o objetivo de apresentar uma cafeteria artesanal de forma elegante e funcional. O site institucional tem como foco a experiência premium do usuário, apresentação detalhada do cardápio e conversão de clientes através de contatos via WhatsApp.

---

## 🛠 Tecnologias

- **Frontend:** React 19, TypeScript, Tailwind CSS 4
- **Routing:** TanStack Router
- **Animações:** Framer Motion
- **Formulários:** react-hook-form + zod
- **Ícones:** Lucide React
- **Deploy:** Vercel

---

## ✨ Funcionalidades

- **10 rotas** completamente funcionais.
- Navegação **SPA** otimizada.
- Seções dedicadas: Home, Cardápio, Categorias, Grãos, Sobre, Contato.
- Formulário de contato com **validação em tempo real**.
- Animações **parallax** no hero.
- **Hover effects** interativos em cards.
- **Toast feedback** para ações do usuário.
- Menu **mobile responsivo**.
- **Reveal animations** ativadas pelo scroll.
- **Design system** baseado em OKLCH.

## 🚀 SEO e Performance

## ⚙️ Como Executar Localmente

```bash
# Clone o repositório
git clone https://github.com/usuario/cafe-aurora

# Entre na pasta
cd cafe-aurora

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## 📦 Build de Produção

```bash
npm run build
```
O projeto compilado será gerado na pasta `dist/`.
